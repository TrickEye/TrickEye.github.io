    CSS

    CSS总结⚓︎

    CSS由以下主要部分组成：选择器和声明（一条或多条）

    最简单的，在html文件的head部分，添加如下：

    <style>
    p {
        color: red;
        font-size: 12px;
    }
</style>

    选择器⚓︎

    选择器用来选择标签，有针对性地设置样式

    基础选择器-标签选择器⚓︎

    语法
    标签名 {
    属性: 属性值;
    属性: 属性值;
    属性: 属性值;
}

    基础选择器-类选择器⚓︎

    html
    <div class='red'> <!--(1)--> 红色类 </div>
    1. 在html中，使用class属性分类
    css
    .red {
    color: red;
}

    多类名：当一个html元素希望属于多个类的时候，在class属性中使用空格来分割多个类名。

    基础选择器-id选择器⚓︎

    类似类选择器，但是在css中用#来开头，id为每个元素独有。

    基础选择器-通配符选择器⚓︎

    在css中使用*来选择，选中所有标签，属性对所有标签生效。

    复合选择器-后代选择器⚓︎

    前后排列两个选择器A,B，用空格隔开，组成后代选择器，后代选择器将选择是A选择器选择出来的元素的后代的，满足B选择器条件的标签。

    举例：ol li { color: blue; }将仅仅对有序列表中的表格元素生效。

    这样的后代选择器可以推广到多个基础选择器的排布。

    复合选择器-子元素选择器⚓︎

    使用>符号链接其他选择器，以选择某元素的最近一级（直接儿子）元素。

    复合选择器-并集选择器⚓︎

    使用,来链接多个选择器，在最终生效时，生效的是这些选择器选择出来的标签集合之并

    复合选择器-伪类选择器⚓︎

    使用:向选择器添加特殊效果，例如:hover，:first-child

    链接伪类

    • a:link 选择所有未被访问的超链接标签
    • a:visited 选择所有已被访问的超链接标签
    • a:hover 鼠标停留在其上的超链接标签
    • a:active 选择鼠标按下未弹起的链接

    Warning

    在使用时，请遵从LVHA的顺序声明，以确保能够正常生效

    :focus伪类选择器，主要针对<input>表单元素，用于选取获得焦点的表单元素。

    属性⚓︎

    字体属性⚓︎

    font-family
    字体系列，举例：p { font-family: '微软雅黑', Arial, Helvatica, sans-serif; }，浏览器顺序匹配第一支持字体。
    font-size
    字体大小，举例p { font-size: 20px; }，对body指定可以指定整个页面的字体大小
    font-weight
    字重，举例.bold { font-weight: bold; }，属性值有以下选项：normal, bold, bolder, lighter, number，系统选择字体支持的最近的字重匹配。数字方面，400=normal, 700=bold
    font-style
    文字样式，p { font-style: normal; }，也有italic可选

    综上，可以使用font来对字体属性整体定义，先后顺序为font: font-style font-weight font-size/line-height font-family;

    文本属性⚓︎

    color
    颜色接受以下几种表述方式：预定义颜色，#RRGGBB，RGB代码rgb(r, g, b)（RGBA代码，这里A为透明度，取值范围为0~1），其中r,g,b为0-255数字或百分比
    text-align
    文本水平对齐方式（left, right, center, justify）
    text-decoration
    不修饰none，下划线underline，上划线overline，删除线linethrough，可以通过修改这个来移除超链接丑丑的下划线
    text-indent
    首行缩进，这里可以用em（当前元素的一个font-size大小）
    line-height
    行高，行间距。修改行高并不会修改字体大小。在有些场合下，通过设置文字的行高同其容器高度，可以实现垂直居中的效果

    元素及其显示方式⚓︎

    Html元素一般分为块元素和行内元素。

    最经典的块元素包括：<h1>, <p>, <div>, <ul>, <ol>, <li>，特点表现为：

    • 独占一行。
    • 高度、宽度、内外边距都可以控制。
    • 宽度默认为父容器的100%。
    • 被视为一个容器，内部可以放置行内元素和块元素（当然，p和hx这类文字类标签并不能放其他标签）

    行内元素/inline element例如：<a>, <strong>, <b>, <em>, <i>, <del>, <s>, <ins>, <u>, <span>特点体现为：

    • 相邻行内元素可以在一行内显示多个。
    • 直接设置其宽和高无效。
    • 默认宽度为其本身的宽度。
    • 只能容纳文本和其他行内元素

    行内块元素：<img/>, <input/>, <td>同时具有块元素和行内元素的特点，表现为：

    • 和相邻行内元素在同一行，但之间会有缝隙，一行可以显示多个（类似行内元素）
    • 默认宽度为其本身内容宽度（类似行内元素）
    • 高度，行高，内外边距可以被控制（类似块级元素）

    在有些情况下，块元素和行内元素两种显示方式可以互相转换，通过CSS选中标签并赋值display属性（inline, block, inline-block）可以转化其显示方式。

    背景⚓︎

    background-color
    背景颜色，可以设置为transparent或者之前提到的颜色值。
    background-image
    背景图像，none或url，url在CSS的写法是background-image: url(images/logo.png);，bgi和bgc可以共存，但是非透明的bgi会挡住bgc
    background-repeat
    图像平铺，在以下取值中：repeat, no-repeat, repeat-x, repeat-y
    background-position
    改变bgi的位置，含有两个由空格区分的参数，分别表示x和y，每个参数可以是具体的数字（百分数/浮点数字加单位）也可以是方位名词（top, center, bottom, left, center, right）只指定一个方位名词的时候另一坐标默认居中
    background-attachment
    背景图像固定，在scroll和fixed之间取值。决定是否跟随页面滚动而滚动

    盒子模型⚓︎

    简要概述：html中的布局元素视为矩形盒子，从内到外，由以下元素组成：

    最内部content，内边距padding，边框border，外边距margin。除了content之外，其他元素都有四个方向（top, bottom, left, right）

    边框⚓︎

    边框分为四个方向，以下的属性值如果提供四个，则会按照顺序应用于上-右-下-左四个边框；如果提供两个，则先后应用于上下和左右；提供三个，则顺序是上-左右-下

    也可以只对一个边框设置属性：例如使用border-top-xxx来设置上面的边框。

    border-width
    边框粗细，边框的宽度是独立于width和height属性的，不会因为边框而压缩盒子的大小
    border-style
    边框样式，取值可以是：none, hidden, dotted, dashed, solid, double, groove, ridge, inset, outset
    border-color
    边框颜色
    border-collapse
    设置border-collapse: collapse;可以合并相邻单元格的边框，多见于表格的单元格边框设置
    border-radius: length
    设置边框的圆角

    内边距⚓︎

    使用padding属性设置内边距大小，内边距不包括在width和height尺寸内，会影响（撑开）盒子的大小，除非没有指定width和height。

    外边距⚓︎

    使用margin属性来设置外边距，外边距也可以设置为auto，盒子有宽度的时候，左右设置为auto可以居中，行内块元素和行内元素的水平居中需要给父元素添加text-align: center;

    外边距合并：当上下两个兄弟关系块元素相遇时，上下距离为两者的上下边距之中更大的值。因此可以只给一个盒子margin值

    外边距塌陷：父子关系块元素同时具有上边距时，父元素将塌陷较大的外边距值。可以为父元素定义上边框，上内边距，或者使用overflow: hidden;属性

    行内元素尽量只设置左右的内外边距

    box-shadow: h-shadow v-shadow blur spread color inset;
    为盒子绘制阴影，这里前两个数字为必选，指出阴影的位置。之后的参数为：模糊距离，阴影尺寸，阴影颜色，是否反转为内部阴影。
    text-shadow: h-shadow v-shadow blur color;
    为文字设置阴影。请参考上一条

    浮动⚓︎

    在此之前介绍的布局被称为标准流。在标准流中，一行显示一个块元素，很难做到块元素居左居右。

    float: left/right;
    通过此属性设置浮动显示，使得不再是一个块独占一行，而是浮动，靠到上一个元素的相应边界。

    浮动的元素将不再占有原有的标准流位置。如果下面还有标准流，会上移到浮动元素的下方。

    这里如何理解浮动呢？可以视为：元素浮起来，来到了一个不同于原有标准流的新层次“浮动层”，然后根据设置的浮动属性，动起来，重新布局找到新的位置。原有的标准流布局的元素在布局的时候完全无视浮动元素，因此自然就会来到浮动元素的下方。

    如果多个盒子设置了浮动，他们会按照属性值在一行内显示，顶端对齐排列。排列没有缝隙，父级宽度不足时另起一行对齐。

    浮动的元素具有行内块特性，如果行内元素被浮动，可以对其指定宽度和高度。

    浮动的元素经常被放在标准流的父级块元素，配合布局使用。

    清除浮动⚓︎

    有些时候我们希望不给父元素设置高度，而是由内部元素自动撑开父元素。但是如果内部元素浮动的话，就会产生不能被撑开的现象。此时就需要清除浮动。清除浮动有诸多方法：

    clear: both
    用于指出在左右两侧都不允许有浮动元素。用于通过某种方式辅助撑开盒子。

    在浮动元素列的最末尾添加空标签：<div style="clear: both"></div>

    为父元素添加样式：overflow: hidden/auto/scroll;

    为父元素添加如下样式 

    .clearfix:after {
    content: "";
    display: block;
    height: 0;
    clear: both;
    visibility: hidden;
}

    为父元素添加： 

    .clearfix:before, .clearfix:after {
    content: "";
    display: table;
}
.clearfix:after {
    clear:both;
}

    定位⚓︎

    标准流和浮动流都不能解决固定一个元素在页面上的显示位置，盖住其他元素这一功能，因此需要定位。定位=定位方式+边偏移

    position: static/relative/absolute/fixed;
    定位方式，分别表示静态、相对、绝对、固定定位。
    top/bottom/left/right: 10px;
    边偏移，定义元素相对于其父元素相应边线的距离。

    默认定位方式，不存在边偏移，按照标准流特性摆放。在布局的时候用的比较少

    移动位置的时候，根据它原来的定位位置来决定。

    不脱标：原来在标准流的位置仍然占有，其他元素根据其原来的位置来排版。

    移动位置的时候，相对其祖籍元素。

    没有祖先元素，或者祖先元素没有定位，则以浏览器为准定位。否则以最亲一级有定位的祖先元素为参考点。

    脱标，在标准流下的位置不保留

    样式表引入方式⚓︎

    内部样式表
    在Html文件中head部分添加<style></style>并在之间写css内容
    行内样式表
    在元素标签内部修改style属性，对此标签生效，<div style="color: red; font-size: 12px;"> something </div>
    外部样式表
    在html中使用<link>标签来引入一个外部的css文件，对html全文生效。<link rel="stylesheet" href="path_to_css">

    CSS特性⚓︎

    层叠性⚓︎

    相同的选择器为标签设置相同的样式，此时一个样式会层叠另一个被冲突的样式，具体采用的样式遵从就近原则。距离此标签最近的一个样式生效。

    继承性⚓︎

    子标签继承父标签某些样式，包括text-, font-, line-, color属性

    一个特别的例子，涉及line-height的继承时，如果采用相对比例（line-height = 1.5表示为1.5个字符高度）

    优先级⚓︎

    不同选择器间有如下的权重：

    选择器权重

    选择器 优先级
    继承或被*选中 0000
    元素选择 0001
    类选择，伪类选择 0010
    ID选择 0100
    行内样式style="" 1000
    !important ∞

    当同时被不同种选择器选中时按照优先级来计算。复合选择器权重叠加。
