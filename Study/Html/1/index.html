    HTML 总结⚓︎

    基本结构⚓︎

    <!DOCTYPE html> <!-- (1) -->
<html lang="en"> <!-- (2) -->
<head>
    <meta charset="UTF-8">  <!-- (3) -->
    <title>title for the html</title> <!-- (4) -->
</head>
<body>
</body>
</html>
    1. 指出文件种类为html
    2. 指出语言为英文
    3. 指出使用字符集为UTF-8
    4. 指出网页的标题

    标签⚓︎

    标题标签成对出现，共有六级，从<h1>content</h1> 到 <h6>content</h6> 字号和层次递减

    段落标签<p>paragraph text </p>指出一段文本

    换行标签<br/> 其实这里不用斜杠的话也可以被正确识别

    文本格式化标签包括：

    • 加粗效果：<strong>加粗效果</strong> 或者 <b>也是加粗效果</b>
    • 倾斜效果：<em>倾斜效果</em> 或者 <i>也是倾斜效果</i>
    • 删除线：<del>删除线</del> 或者 <s>也是删除线</s>
    • 下划线：<ins>也是下划线</ins> 或者 <u>也是下划线</u>

    div标签<div>div content</div>占据一整行，与此相比，span标签<span>span content</span>排布比较紧密

    图像标签img例如：<img src="picture.jpg" alt="替换文本" title="标题" width="640" height="480" border="10"/>，有如下属性

    img标签属性

    src

    必选，一个链接指向要显示的图片（相对、绝对（不推荐）和网络地址均可）

    alt

    替换文本，当图片不能正常显示的时候，使用替换文本来放在图片的位置

    width/height

    高度/宽度，像素数量或百分比大小

    border

    边框宽度（像素数量）此属性将被弃用，现在通常使用CSS来设置此属性

    超链接hyperref例如：<a href="https://baidu.com" target="_blank">blank</a>，有如下属性

    href标签属性

    href

    超链接指向的内容，可以是相对，绝对，网络地址，从用途分类也可以分为以下几类

    • 常规链接 https://baidu.com，示例<a href="https://baidu.com" target="_blank">demo</a>
    • 空链接 #，示例<a href="#" target="_blank">demo</a>，点击的时候跳转的仍然是本页面
    • 文件链接 这个就不举例了，点击的时候将会触发浏览器的下载
    • 锚点链接 #id，这里这个id是由其他标签在属性中添加id="dist"实现索引的1，示例<a href="dist" target="_blank">demo</a>
    target

    点击超链接后的行为，可选

    • _blank：在新的标签页中打开
    • _self：覆盖自身
    • 此外的属性值还有_parent，_top用的比较少

    表格⚓︎

    表格由标签<table></table>指出和围绕。在表格内使用<tr></tr> <!--table row-->指出行，由<td></td>指出单元格数据。

    表头由标签<th></th>指出，他与td标签有同样的层次，但是会加粗表示。

    为了更明显的区分表头和表身，可以使用标签<thead></thead>和<tbody></tbody>分别环绕。

    单元格的合并⚓︎

    合并包括行合并和列合并。行合并需要在td/th标签中加入colspan="x"属性，x表示要合并的单元格数，随后删除被合并的单元格

    列合并需要加入rowspan="x"属性，随后删除被合并的单元格

    表格标签的其他属性⚓︎

    其他属性包括：align="(center,left,right)"，border="x"，cellpadding="x"指单元格内容距离单元格边框的距离，cellspacing="x"指单元格之间互相的距离，width/height="x"

    列表⚓︎

    列表包括以下几种：无序列表，有序列表，自定义列表

    无序列表和有序列表标签不同，分别使用<ul></ul>和<ol></ol>围绕，而表格内部的单元都是使用<li></li>围绕的

    自定义列表比较类似material for mkdocs中的解释列表，或者\(\LaTeX\)的description环境。使用<dl></dl>围绕，内部单元格涉及词条<dt></dt>和解释<dd></dd>

    表单⚓︎

    表单域指标签对<form></form>，这对标签围绕的空间为表单域，在提交的时候将会一并将输入的数据提交到对应的服务器。<form>标签具有以下属性：action="url"指出接受数据的服务器，method="(get, post)"指出接收方式，name指出表单域名称

    输入标签<input></input>有多种种类。用type属性来指出。

    1. input标签的value属性在text/password类型的输入框中将会呈现为缺省的文本，在选择框中体现为向后台反应的结果（不会在前端显示）maxlength属性指出了最大的长度，超过此长度的输入会被舍弃。

    4. 在单选和多选可以添加checked属性，在一开始打开界面的时候就标记为选中

    标签⚓︎

    <label></label>标签用于绑定一个表单的元素，以实现当点击label包围的文本时，将焦点转到对应的表单元素上。通过label标签的for属性和input标签的id来实现索引。

    <label for="element_id">用户名</label> <input type="text" id="element_id">

    Html5的新东西⚓︎

    新的语义化标签⚓︎

    html5在原有的div基础上提供了更具有语义化的标签，而不再需要通过为div设置class来区分之间的语义。对于搜索引擎来说也可以精准的理解标签的语义。<header>,<nav>,<artivle>,<section>,<aside>,<footer>

    新的媒体标签⚓︎

    <audio>,<video>标签可以在页面中插入音频和视频。video标签主要支持MP4格式。语法如<video src="path_to_video" controls="controls"></video>

    • src="path_to_video"指出要播放的视频位置。
    • autoplay自动播放。但是在chronuim内核需要配合mute才能自动播放。
    • muted静音
    • controls向用户显示播放控件
    • loop是否循环播放
    • preload="auto/none"预先加载视频或不加载视频
    • poster="img_url"加载等待的时候的画面图片

    一种兼容性比较好的写法是： 

    <video width="320" height="240" controls>
    <source src="movie.mp4" type="video/mp4">
    <source src="movie.ogg" type="video/ogg">
    你的浏览器不支持 video 标签
</video>

    • autoplay自动播放
    • controls向用户显示播放控件
    • loop循环
    • src要播放的音频
    <audio controls>
    <source src="horce.ogg" type="audio/ogg">
    <source src="horce.mp3" type="audio/mpeg">
</audio>

    html5对input类型进行了扩充，包括：

    属性值 说明
    email 限制输入为邮箱
    url 限制输入为url
    date 限制输入日期
    time 限制输入时间
    month 月份
    week 周
    number 数字
    tel 手机号码
    search 搜索框
    color 选择颜色

    此外，对input表单新增了required属性，当具有required="required"属性时，提交时不能为空。

    新增placeholder属性，当placeholder="something"时，输入框呈现灰色的something，输入任何内容后消失。

    新增autofocus属性，当具有此属性时，页面被加载时自动将光标置于此输入框。

    新增autocomplete属性，默认基于输入记录补全，但是通常使用此属性=off来关闭，避免隐私泄露。

    新增multiple属性，可以使输入的文件可以多选。

    1. 所以其实任何一个标签都应当有一个可选的属性叫做id，通过对id的赋值可以编写CSS，使用锚点链接等。 ↩
