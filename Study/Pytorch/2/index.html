    Pytorch安装与入门⚓︎

    安装⚓︎

    如果你财力雄厚，电脑里面有一张可爱的N卡，甚至还能跑CUDA，那快去这里下载，pytorch支持很多种包管理器，比如，要下载一个运行在windows系统，适用于Python语言，通过pip包管理器来安装，外加CUDA11.6的Pytorch，对应的命令是：

    pip3 install torch torchvision torchaudio --extra-index-url https://download.pytorch.org/whl/cu116

    下载过程太慢的请参考如何为pip换源

    安装过程中有必要为Windows开启长路径支持，请参考https://pip.pypa.io/warnings/enable-long-paths

    安装情况校验

    张量 Tensor⚓︎

    张量是一个很大的概念，按阶数分，张量可以分为：{0阶张量：标量和常数；1阶张量：向量；2阶张量：矩阵；...N阶张量}

    在python输出的结果里面，有几层方括号/方括号的最大深度，Tensor 就有多少阶

    Tensor 的创建⚓︎

    torch.tensor([[1., -1], [1., -1]])

tensor([[ 1., -1.],
        [ 1., -1.]])

    import numpy as np
torch.tensor(np.array([[1, 2, 3], [4, 5, 6]]))

tensor([[1, 2, 3],
        [4, 5, 6]], dtype=torch.int32)
    • torch.empty(3,4)创建3行，4列的空tensor，使用无用数据填充，所以你不应当相信它会被初始化为0
    • torch.ones([3, 4])创建3行4列的全1张量，请注意列表括号
    • torch.zeros([3, 4])创建3行4列的全0张量
    • torch.rand([3, 4])创建3行4列全随机张量，值的取值范围为[0,1)
    • torch.randint(low=0, high=10, size=[3, 4])创建3行4列全随机整数张量，值的取值范围为[low, high)
    • torch.randn([3, 4])创建3行4列随机张量，值的分布遵循标准正态分布

    torch.tensor与torch.Tensor

    两者的一个区别在于调用torch.tensor(x)，（x为一个数字）的时候。Tensor()认为这个数字描述了他的形状，使用随机的数字来填充这个张量，只有在传入一个可迭代对象的时候，Tensor()会认为这是数据。

    torch.tensor(2)
    tensor(2)
    torch.Tensor(2)
    tensor([1.0415e-37, 4.5907e-41])

    Tensor 方法和属性⚓︎

    item()
    对于任意阶，只有一个元素的 Tensor 返回那个唯一的元素。如果元素不唯一的，报错。
    numpy()
    转换为numpy数组
    size()
    获取这个 Tensor 的形状。当含参的时候，返回对应位置的形状数字，也可以像python列表那样使用负数从后往前取值
    view()
    接受几个参数（或者把这些参数写进一个元组/列表），转换 Tensor 到对应的形状，浅拷贝，仅仅发生形状改变。如果一个参数为-1，则此参数对应的尺寸采用自适应
    dim()
    获取阶数
    max() / min() / std()
    获取最大值，最小值，标准差，以一个仅有唯一元素的 tensor 来反映。std()只支持内容为浮点的对象，通过t1.float()来转换
    t()
    转置。只能转置一个小于等于2阶的 Tensor，大于此阶数的需要用 transpose()指出要转置的维度，或者permute()来指定维度的交换顺序。例如，transpose(0,1)与permute(1,0,2)是等价的

    Tensor 中数据的访问和赋值使用 t[1,2,3] 来指出各个维度的索引。也可以使用t[:,1]来进行切片

    Tensor 数据类型⚓︎

    数据类型 tensor.dtype tensor type
    32b 浮点 torch.float32 | torch.float torch.*.FloatTensor
    64b 浮点 torch.float64 | torch.double torch.*.DoubleTensor
    16b 浮点 torch.float16 | torch.half torch.*.HalfTensor
    8b uint torch.uint8 torch.*.ByteTensor
    8b int torch.int8 torch.*.CharTensor
    16b int torch.int16 | torch.short torch.*.ShortTensor
    32b int torch.int32 | torch.int torch.*.IntTensor
    64b int torch.int64 | torch.long torch.*.LongTensor

    通过t.dtype获取 Tensor 的数据类型。在使用torch API创建 Tensor 的时候也可以指定数据类型，或者使用最右边这一列的类型来实例化

    要产生类型转换，可以使用t.int()

    运算⚓︎

    相同形状的 Tensor 可以被使用+符号相加，也可以使用torch函数 t1.add(t2)，这个函数不会造成t1的修改，相应的，t1.add_(t2)会就地修改t1。

    当然了，在很多函数后面加下划线都可以就地修改调用此方法的 Tensor
